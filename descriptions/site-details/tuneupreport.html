A system, written internally for Hobsons, whose control and maintenance were transferred to me. The Tune-Up Report tool provides numeric data about product use to the Client Development team without them needing access to or needing to query the database. A date range is supplied, which allows results from different periods to be analyzed along with allowing comparison with all-time data. I also added a multi-client version of this tool so that data from across the organization can be combined and averaged.
