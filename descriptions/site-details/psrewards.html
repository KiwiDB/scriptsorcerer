An internal interface and management system I built for Hobsons. Used to track "reward points" given to employees by managers based on performance. Interface allows for awarding points and managing users; the display is integrated into our Jive-based intranet site and displays the top-five points earners as well details about the points awarded to any employee. Jive user profile images are used throughout the display.
