Lead Submission

Technologies

  • Systems PHP cURL
  • Languages HTML5 (appcache and localStorage), CSS3 (responsive), Javascript, XML
An offline-available web application I wrote for the Media Services and Student Recruitment teams at Hobsons. The system allows leads to be submitted to the system of a client at student events, either by the students themselves or by members of our team. The application uses HTML5 local storage to store leads locally, which are mass-submitted after the event is over. HTML5 application cache is used to allow access to the system when an internet connection is not available; when offline, leads are saved locally and submission becomes possible when internet access is restored.
