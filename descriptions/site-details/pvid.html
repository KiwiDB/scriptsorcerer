Personalized Videos

Video Generation Software

  • My Role Head Developer, Mobile System Designer

Technologies

  • Systems Flash, PHP "GD", AWS Host, AWS S3, AWS SQS
  • Languages HTML5, CSS3 (responsive), Javascript, PHP (with GD), jQuery, AJAX, ffmpeg
A new framework for a video personalization system from Hobsons, expanded to include mobile support (my involvement in the project was the design and development of the new framework and the mobile component of the system). The videos are personalized by inserting some personal information into placeholders (see the screenshots for examples). The system uses Flash when possible, and switches to the mobile component if Flash support is not detected. The mobile system uses PHP's GD library to render text on frames when necessary, and the ffmpeg app to create an MP4 video from the frame sequences. The system runs on AWS server pools and uses an AWS SQS queue to control the workflow, and an AWS S3 server for long-term storage of videos once rendered.
