An internal system I wrote for Hobsons, integrated into our Jive-powered intranet. Provides an interface for Implementation Specialists to insert on-going data about implementations, and a company-wide system for browsing or searching these projects. The whole system (fields to save for projects, how they display in the system, requirements for a project to fit into a category, etc) is customizable by user interfaces made for non-technical individuals. Also sends go-live emails to all who subscribe to receive notifications.
