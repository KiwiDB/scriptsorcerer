Interest Page Creator

Task Automation

  • My Role Project Manager, Developer

Technologies

  • Systems SimpleBrowser, Excel, MySQL
  • Languages HTML5, CSS3 (responsive), Javascript, PHP, MySQL, jQuery
A large application used to automate the creation of interest forms in Connect, a Hobsons product sold to colleges and universities. Replaces an existing, inefficient interface with a new jQuery and AJAX-based interface. Begins at one of three entry points -- start the new interest form from the default system fields, start from an Excel document containing details about the new form, or restore from a form already saved in the system. Loads into a GUI where fields can be added, removed, or edited (including data validations and UI widgets), along with setting general information about the form and submission response email. The application communicates the form details to Connect, and creates the necessary XML, CSS and Javascript.
