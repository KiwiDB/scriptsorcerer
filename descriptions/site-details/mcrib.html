"McRib" Monitor

Performance Monitor

  • My Role Developer, Designer

Technologies

  • Systems IMAP, XML
  • Languages HTML5, CSS, Javascript, PHP, AJAX, XML
An monitor I wrote for Hobsons. "McRib" is the name given to a piece of our products that handles email tracking. The monitor watches for any of the "McRib" processes to go down and sends alert emails if they do. The monitor works by accessing specialized email accounts created for it, and reading, deleting and sending emails each hour. I also created an interface allowing for changing configurations for the monitor and seeing graphed information about the performance of the "McRib" processes.
