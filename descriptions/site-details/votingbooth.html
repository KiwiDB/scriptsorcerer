Voting Booth

Jira Integration

  • My Role Developer, Content Implementation

Technologies

  • Systems Jira Web Services, MySQL
  • Languages HTML5, CSS3 (responsive), Javascript, PHP, SOAP
An application written to aid in task prioritization internally at Hobsons. Integrating with Jira Issue Tracker, the system is based around voting outstanding issues up-and-down. While this is functionality that exists in Jira itself, the Voting Booth interface is meant to encourage used of these features by providing a custom experience built for the needs of the company.
Project Type Hobsons Automation
